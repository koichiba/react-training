# simple-event-creator

## Overview

- イベント名を入れるだけで専用サイトが自動で作れる Web アプリケーション(詳細も入力化)
- PWA を使用することで、作ったイベントサイトをアプリっぽく使用できる
- プッシュ通知でイベントの情報をリアルタイムにユーザに届けられる
